# 💻 Personal Website

This is the repo for my personal website containing a charts page (listing my favourite songs), a public study time tracker which I can control via a private dashboard, and a URL shortener which I can use to mask/shorten long links before sharing.

## How to Access
Use the live demo at the bottom of the README
### OR
- Download the project files as a `.ZIP` or clone the repo using `git clone https://github.com/Matthew-Seaber/personal-website.git` in your terminal
- Run `npm install` to install the required dependencies
- Run `npm run dev` to start a local development server (default: `http://localhost:3000`)

Feel free to create a pull request if you'd like to contribute or have any suggestions for new features!

## Features
- Charts page listing my favourite songs
- Public study/work time tracker controlled by a private dashboard
- URL shortener to shorten long links before sharing
